class opendp.smartnoise.synthesizers.mwem.MWEMSynthesizer(Q_count=400, epsilon=3.0, iterations=30, mult_weights_iterations=20, splits=[], split_factor=None, max_bin_count=500, custom_bin_count={})¶

Bases: opendp.smartnoise.synthesizers.base.SDGYMBaseSynthesizer

N-Dimensional numpy implementation of MWEM. (http://users.cms.caltech.edu/~katrina/papers/mwem-nips.pdf)

From the paper: “[MWEM is] a broadly applicable, simple, and easy-to-implement algorithm, capable of substantially improving the performance of linear queries on many realistic datasets… (circa 2012)…MWEM matches the best known and nearly optimal theoretical accuracy guarantees for differentially private data analysis with linear queries.”

Linear queries used for sampling in this implementation are random contiguous slices of the n-dimensional numpy array.

mwem()¶

Runner for the mwem algorithm.

Initializes the synthetic histogram, and updates it for self.iterations using the exponential mechanism and multiplicative weights. Draws from the initialized query store for measurements. :return: A, self.histogram - A is the synthetic data histogram, self.histogram is original histo :rtype: np.ndarray, np.ndarray

sample(samples)¶

Creates samples from the histogram data. Follows sdgym schema to be compatible with their benchmark system.

NOTE: We are sampleing from each split dimensional group as though they are independent from one another. We have essentially created len(splits) DP histograms as if they are separate databases, and combine the results into a single sample.

N samples
