class opendp.report.Report(results=None)¶

Bases: object

A differentially private report

Represents a list of result objects, with each result being a vector of differentially private results sharing common source and privacy paramaters.

The individual result objects need not be the same length, though lengths will be the same for multi-column SQL outputs.
